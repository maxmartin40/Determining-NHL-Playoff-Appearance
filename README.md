# Determining-NHL-Playoff-Appearance

This is a personal side project that uses the "NHL Team Data Per Season + Playoff Appearances" dataset created by Ryanhdar on Kaggle. The dataset can be accessed [here](https://www.kaggle.com/datasets/ryanhdar/nhl-team-data-and-playoff-result).

More information and visuals will be added to this page as the project moves along.
